The Attention Mechanism · Session 2 · Mix = weighted blend of values

Attention: a token
reaches across the sentence.

A token scores every other token by alignment, focuses those scores into a soft distribution with softmax, then mixes the tokens' values in those proportions. This demo isolates that mechanism end-to-end on a real sentence — driven by GPT-2's actual attention across all 12 layers and heads, or by GloVe word vectors as a transparent stand-in. Click ⤢ Expand to blow the panel up for the room.

Query — what a token is looking for Key — what a token offers Value — what gets moved

AMix — a weighted blend of values

The point: the output at a position is the attention-weighted average of the value vectors, outi = Σj aij vj — a soft lookup. A causal mask blocks the future so a token can only read what came before it. With real GPT-2 weights, the same machinery produces recognisable head behaviours — previous-token heads, attention sinks, induction.

⚠What we're simplifying

  • GloVe mode uses identity projections on static word vectors. We set WQ=WK=WV=I and use GloVe vectors, so attention there is driven by raw word similarity. Real heads learn WQ,WK,WV (the “randomize” button hints at this), operate inside the residual stream, and add an output projection WO. The GPT-2 (real) mode shows the genuine learned attention.
  • One head at a time. Real layers run many heads in parallel (the coda), and the interesting behaviour — induction, in-context learning — comes from composing attention across layers.
  • LayerNorm and the MLP are out of frame. This demo isolates the attention sub-layer; the residual-stream panel (GPT-2 mode) shows how a token's vector is reshaped as context folds in, but the full block has more moving parts.
