# Pilot class repo

Summer pilot of *AI Coding Agents for Engineering Research*, a graduate course
I'm developing for the department. The pilot runs the course material with six
students from the lab to test what works and what doesn't before the full Fall 2026 offering.

This repository hosts the class landing page. It links out to each student's
portfolio site, which they build and maintain in their own repository in the
[AI-for-engineering-research](https://github.com/AI-for-engineering-research)
organization.

## How it's organized

- Each student owns a private repo in the org for their project and portfolio.
- This repo holds the shared landing page that links to every portfolio.
- Course material and instructor notes live alongside the landing page here.

## For students

1. Create your own repo in the org for your project.
2. Build a portfolio site in it using whatever tooling you like.
3. Don't settle for simple — push the boundaries on everything, even the HTML,
   interactive elements, and color schemes. You'll get a feel for what the AI
   models can do well, and develop an intuition for where they fall short.

Keep your work attributed and reproducible — recording how you used AI agents is
part of the course, not an afterthought.
